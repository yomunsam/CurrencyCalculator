# 新增货币类型指南（快速版）

本文档用于指导开发者以最小改动新增一个货币类型（法币或加密币）。

## 1. 在 `CurrencyCatalog` 注册新货币

文件：`Src/CurrencyCalculator.Web/Core/CurrencyCatalog.cs`

- 法币：新增一行 `Fiat(...)`
- 加密币：新增一行 `Crypto(...)`

示例（法币）：

```csharp
Fiat("SGD", "sg", 2, "Singapore Dollar", "新加坡元", "シンガポールドル"),
```

字段说明：

- `Code`：ISO 货币代码（建议大写）
- `Icon`：法币使用 2 位国家/地区代码（用于 flagcdn），加密币使用符号
- `DisplayDecimals`：前端展示小数位
- `Names`：需覆盖 `en-US`、`zh-CN`、`ja-JP`

## 2. 更新 fallback 生成脚本

文件：`.github/scripts/update-fallback.ps1`

将新货币代码加入 `$supportedCodes`，否则 GitHub Actions 生成的 fallback 不会包含该币种。

## 3. 刷新 fallback 数据文件

在仓库根目录执行：

```powershell
pwsh .github/scripts/update-fallback.ps1
```

会更新：`Src/CurrencyCalculator.Web/wwwroot/fallback/latest-rates.json`

## 4. 最小验证

在仓库根目录执行：

```bash
dotnet build Src/CurrencyCalculator.Web/CurrencyCalculator.Web.csproj -c Release
```

验证点：

- 页面可选中新增币种
- 实时汇率与 fallback 均可返回新增币种汇率
